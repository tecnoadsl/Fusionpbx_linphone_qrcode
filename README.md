# Linphone QR Code per FusionPBX

App per generare QR code per configurare automaticamente l'app Linphone.

## Installazione

```bash
# Come root
unzip linphone_qrcode.zip
cd fusionpbx_linphone_qrcode
chmod +x install.sh
./install.sh
```

## Dopo l'installazione

1. Accedi a FusionPBX come **superadmin**
2. Vai su **Advanced > Upgrade**
3. Clicca **Menu Defaults**
4. Clicca **Permission Defaults**
5. **Logout** e **Login**

## Menu

**Accounts > QR Code Linphone**

## Disinstallazione

```bash
chmod +x uninstall.sh
./uninstall.sh
```

Poi vai su Advanced > Upgrade > Menu Defaults
